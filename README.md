# Neon

Gaze analysis with interactive velocity plot for Neon eye-tracking recordings.

**This project is powered by AI** (e.g. Cursor) for development and documentation.

## Setup

- Python ≥ 3.10
- Install with: `uv sync` (or see `pyproject.toml` for dependencies)

## Dataset

See [dataset/README.md](dataset/README.md) for how recordings are produced and how the dataset directory is structured.
